# Sales-Performance-Analysis
Data Analytics project involving collection , cleaning and modelling of data to uncover key business KPI's and extract insights for the stakeholder's to derive business intelligence and data driven decisions

## Project Overview

This project analyzes sales performance data using Excel, MySQL, Power BI, and Tableau.

The objective was to identify:

- Top performing products
- Regional sales trends
- Customer behavior patterns
- Revenue growth opportunities
- Customer Analysis
- Product & Regional Analysis
- Time Series Analysis
---

## Tools Used

- Excel
- MySQL
- Power BI
- Tableau

---

## Project Workflow

1. Data Collection
2. Data Cleaning
3. SQL Analysis
4. Dashboard Development
5. Business Insights & PoerPoint Presentation

---

## Key KPIs

- Total Sales
- Total Profit
- Profit Margin
- Production Cost
- Average Order Value

---

## Business Insights

- Bikes Stands out the top Category contributing to more than 90 % of the sales
- Product Category Bike generated highest profit.
- Holiday Seasons i,e July & Decemder shows the highest revenue
